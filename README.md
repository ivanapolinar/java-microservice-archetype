# java-microservice-archetype

Este repositorio define un **arquetipo de microservicio Java Spring Boot**
para tu organización. La idea es usarlo como referencia en un GitHub Copilot
Space dedicado a la creación de nuevos microservicios.

Incluye:

- Estructura de paquetes recomendada.
- Configuración base de Spring Boot (Web + Validation).
- Modelo estándar de respuesta (ApiResponse / ErrorResponse).
- Manejador global de errores de ejemplo.
- Configuración de pruebas (JUnit 5) y JaCoCo en Maven.
- Documentación para que Copilot genere nuevos servicios siguiendo este patrón.
