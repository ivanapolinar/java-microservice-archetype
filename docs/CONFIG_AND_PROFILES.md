# Configuración y perfiles

El arquetipo incluye:

- `application.yml` con configuración básica.
- Perfiles recomendados:
  - `dev`: para desarrollo local.
  - `test`: para pruebas automatizadas.
  - `prod`: para producción (ejemplo mínimo).

Puedes extender estos archivos para:
- Configurar fuentes de datos.
- Configurar logs.
- Integrar con service discovery, etc.
